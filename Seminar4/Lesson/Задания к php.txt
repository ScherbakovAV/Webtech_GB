Продолжаем работать в том же файле с первого задания.
 
1. Необходимо сформировать массив с данными для блока "Навыки", массив должен содержать значения навыков и процент владения ими. Пример массива:
$skills = [
    [
        'name' => 'Фотошоп',
        'percent' => 30,
    ],
    [
        'name' => 'Иллюстратор',
        'percent' => 70,
    ],
    [
        'name' => 'Медиа',
        'percent' => 40,
    ],
];
2. Вывести данные из массива на страницу.
